# Установка и запуск

Инструкция для тех, кто впервые ставит Julia. Весь процесс — 5 минут.

## 1. Установите Julia

Самый надёжный способ — через официальный установщик **juliaup**.

**Windows** (PowerShell):
```powershell
winget install julia -s msstore
```
Если `winget` недоступен, скачайте установщик вручную: https://julialang.org/install/

**macOS / Linux**:
```bash
curl -fsSL https://install.julialang.org | sh
```

После установки откройте **новое** окно терминала и проверьте:
```bash
julia --version
```
Должно вывести что-то вроде `julia version 1.11.x` (нужна версия 1.9 или новее).

## 2. Скачайте проект

```bash
git clone https://github.com/Darlakec/BrainMath.git
cd BrainMath
```

Если у вас не установлен `git`, можно просто скачать ZIP-архив кнопкой **Code → Download ZIP** на странице репозитория и распаковать его.

## 3. Установите зависимости (одна команда)

В папке проекта запустите:

```bash
julia --project=. -e "using Pkg; Pkg.instantiate()"
```

Эта команда читает файл `Project.toml` и сама ставит нужную версию `GLMakie` (библиотека для графики/окна) — ничего вводить вручную не нужно. Первый раз может занять 3–10 минут, т.к. `GLMakie` тянет за собой графический стек (OpenGL-биндинги).

## 4. Запустите программу

```bash
julia --project=. Main.jl
```

Откроется окно с графиками и панелью управления.

> **Важно:** ключ `--project=.` обязателен — без него Julia не увидит зависимости из `Project.toml`. Если лень печатать его каждый раз, запустите `julia --project=.` без файла, а потом внутри REPL: `include("Main.jl")`.

## Частые проблемы

**"GLMakie" не открывает окно / ошибка про OpenGL / "no display"**
Нужна видеокарта/драйверы с поддержкой OpenGL 3.3+. На сервере без графического окружения (например, чистый Linux по SSH) GLMakie не запустится — нужен локальный компьютер или X11-форвардинг.

**Установка зависает на "Precompiling"**
Это нормально для первого запуска — Julia компилирует пакеты. Подождите, последующие запуски будут быстрыми.

**Linux: не хватает системных библиотек (libGL, libXrandr и т.п.)**
```bash
sudo apt install libgl1 libxrandr2 libxinerama1 libxcursor1 libxi6
```

**Хочу обновить Julia**
```bash
juliaup update
```
